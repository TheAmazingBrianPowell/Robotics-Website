Robotics Website Code for Roseburg High School's FTC teams

## About
The Robotics Website was created to show Roseburg High School's outreach in the for their FIRST Tech Challenge Teams, it is currently still in the development process and still lacks content. Placeholder text and content are already set. The website uses HTML, CSS, Javascript, and PHP and is currently hosted at https://rhs-robotics.herokuapp.com
